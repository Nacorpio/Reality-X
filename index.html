What is Reality-X?

Reality-X is a simulation of the real-life universe and society. It's made out of APIs and projects linked together to one large project. There are many APIs, including the Economy API and the Traffic API.

What is the point of this project?

Firstly, it's a lot of fun to be working on a project that simulates real life. Secondly, there are many good uses of an engine like this. For example, it can be used to see what happens if one small change is made to the economy. We will be adding a Physics-, Mechanics- and Mathematics API. Those will all make sure that this project is as realistic as possible. We're planning on making this into something that friends can do for fun as well.
